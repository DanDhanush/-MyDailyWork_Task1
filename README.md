# Number Guessing Game (Java)

## Description
This is a simple Java console-based number guessing game.
The program generates a random number, and the user tries to guess it.

## Features
- Random number generation
- User input handling
- Feedback (Too high / Too low / Correct)
- Limited number of attempts (optional)
- Score tracking (optional)
- Multiple rounds (optional)

## How to Run
1. Compile the program:
   javac NumberGame.java

2. Run the program:
   java NumberGame

## Example
Enter your guess: 50  
Too low! Try again.  

Enter your guess: 75  
Too high! Try again.  

Enter your guess: 63  
Correct! 🎉
